# 予約導線つきAI診断LPビルダー（MVP）

LINEリストの見込み客を、診断によって予約に近づける「予約転換ツール」です。
追加インストール不要（Python標準ライブラリのみ）で動きます。

## 起動方法

```bash
cd 診断ツール
python3 server.py
```

ブラウザで **http://localhost:4173/** を開きます。停止は `Ctrl+C`。

データは `data/db.json` に保存されます（サーバDB）。初回起動時にサンプル診断
「30代から痩せない原因診断」が自動で入ります。

## 構成

| ファイル | 役割 |
|---|---|
| `server.py` | バックエンド（静的配信＋JSON REST API、ファイルDB） |
| `index.html` / `app.js` / `styles.css` | フロントエンド（管理画面・公開診断ページ） |
| `data/db.json` | データ保存先（診断・計測イベント） |

## 主な機能

- **管理画面**：診断一覧／作成・編集（質問・スコア・結果タイプ・CTA）／✨AI下書き生成／プレビュー／公開・非公開
- **公開診断ページ**：スマホファースト、1問ずつ・進捗バー・戻る → 点数型タイプ診断 → 結果ページ（10構成）→ 結果タイプ別の予約CTA
- **計測**：診断開始数・完了数・完了率・予約クリック数・予約クリック率・結果タイプ別人数・流入元別
- **URL**：公開URL `#/d/{slug}`、流入元別 `?source=line` などを発行

## 別の端末（スマホ）から開くには

`server.py` は `0.0.0.0` で待ち受けるため、**同じWi-Fi（LAN）内**なら
PCのローカルIPでアクセスできます。

```text
http://<PCのIPアドレス>:4173/#/d/diet-30?source=line
```

PCのIP確認（macOS）:
```bash
ipconfig getifaddr en0
```

### LINE配信で外部に公開する場合（本番）

インターネットからアクセスできる必要があるため、いずれかが必要です。

- **トンネル**（手軽・検証向け）：`cloudflared` や `ngrok` で一時的な公開URLを発行
- **デプロイ**（本番向け）：Render / Railway / Fly.io などにこのフォルダを配置し
  `python3 server.py` を起動（`PORT` 環境変数で待ち受けポートを指定可能）

## 注意（MVPの制約）

- ログイン・会員権限は未実装（単一管理者前提）
- AI生成は外部API未接続のローカルテンプレ合成（必要ならClaude API連携に差し替え可能）
- 表現は薬機法等に配慮し「可能性があります／個人差があります」等の柔らかい表現を採用
